# 🐍 Python Basics

A simple repo to learn core Python concepts through practical code examples.
Each file covers one topic — read the `# comments` inside for key explanations.

---

## 📁 Files

| File | Topic |
|------|-------|
| `basic_code.py` | Variables, data types, operators, I/O |
| `conditions_loop.py` | if/elif/else, for, while, break, continue |
| `python_functions.py` | Functions, args, kwargs, return, scope |
| `python-ds.py` | Lists, tuples, sets, dictionaries |
| `clasess_oop.py` | Classes, objects, `__init__`, attributes |
| `oop_class_methods.py` | Class methods, static methods, properties |
| `oop_inheritance.py` | Inheritance, super(), method overriding |
| `multithreding.py` | Threads, threading module, concurrency basics |
| `request_os_module.py` | `os` module, `requests` library, file paths |

---

## 🚀 How to Use

```bash
# Clone the repo
git clone https://github.com/your-username/python-basics.git
cd python-basics

# Install dependencies (only needed for requests)
pip install -r requirment.txt

# Run any file
python basic_code.py
```

---

## 💡 Comment Guide

Inside every file, look for these comments:

```python
# 📌 IMPORTANT — key concept, don't skip this
# 💡 TIP       — best practice or shortcut
# ⚠️  GOTCHA   — common beginner mistake
```
